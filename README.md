# VITyarthi Digital Literacy Project

## 📌 Overview
This project focuses on building digital literacy skills through practical tasks such as creating awareness resources, developing a digital portfolio, exploring coding platforms, practicing professional communication, and understanding cybercrime.

---

## 📂 Project Structure

### 🔹 Task 1: Digital Literacy Infographic
Created a one-page infographic explaining digital literacy, useful tools, safe internet practices, and email etiquette.

📁 Folder: task-1-presentation/

---

### 🔹 Task 2: Student Digital Portfolio
Developed professional profiles on platforms like GitHub, LinkedIn, and Kaggle to build an online presence.

📁 Folder: task-2-portfolio/

---

### 🔹 Task 3: Coding & Collaboration Platforms
Completed a coding challenge on HackerRank and created a Google Form quiz for digital literacy awareness.

📁 Folder: task-3-platforms/

---

### 🔹 Task 4: Email & Etiquette Guide
Drafted professional emails and created a social media do's and don'ts checklist.

📁 Folder: task-4-email-etiquette/

---

### 🔹 Task 5: Cybercrime Awareness
Prepared a case study on phishing and created a prevention checklist for online safety.

📁 Folder: task-5-cybercrime/

---

## 🎯 Objectives
- Understand digital literacy concepts  
- Build a professional digital presence  
- Improve coding and collaboration skills  
- Learn professional communication  
- Create awareness about cyber safety  

---

## 🛠️ Tools Used
- Canva (Infographic Design)  
- GitHub (Project Hosting)  
- Google Forms & Sheets  
- HackerRank  
- MS Office  

---

## 📌 Conclusion
This project helped in developing essential digital skills, improving online safety awareness, and enhancing professional communication. It provides a strong foundation for academic and career growth in a digital environment.
